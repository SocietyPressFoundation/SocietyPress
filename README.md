# SocietyPress

**Commercial WordPress plugin for genealogical societies, historical societies, and heritage organizations.**

A valid license key is required. [Purchase at charlesstricklin.com/societypress](https://charlesstricklin.com/societypress)

---

## Overview

Membership management built for the volunteers who actually run these organizations: dues and renewals, searchable member directories, committee management, and governance tracking.

Senior-friendly by default. No subscriptions required to use your own data. Your society, your server, your control.

## Features

- Member database with custom fields
- Dues tracking and renewal reminders
- Member directory (public or members-only)
- Committee and role management
- Governance document library
- Senior-accessible UI (large touch targets, high contrast, no swipe gestures)

## Requirements

- WordPress 6.0+
- PHP 8.0+
- MySQL 5.7+ or MariaDB 10.3+
- Valid SocietyPress license key

## Installation

1. Purchase a license at [charlesstricklin.com/societypress](https://charlesstricklin.com/societypress)
2. Download the plugin from your account
3. Upload to `/wp-content/plugins/societypress-core/`
4. Activate in WordPress admin
5. Enter your license key in Settings > SocietyPress
