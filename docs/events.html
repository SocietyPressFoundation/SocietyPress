Events System

SocietyPress includes a full-featured events system for managing your society's classes, workshops, meetings, and other gatherings. Events can have registration, multiple time slots, waitlists, and more.

Creating an Event

  1. Go to SocietyPress → Add New Event
  2. Enter the event title
    Give your event a clear, descriptive name (e.g., "Beginning Genealogy Workshop" or "Monthly Meeting - January").
  3. Add the description
    Use the editor to describe what the event is about. You can add formatting, images, and links.
  4. Set event details
    In the "Event Details" box on the right:
    • Date — When the event takes place
    • Start time — What time it begins
    • End time — What time it ends
    • Location — Where it's held (e.g., "Main Meeting Room")
    • Address — Full address if needed
    • Instructors — Who's leading the event
  5. Choose an event type
    Select a category (Class, Workshop, Meeting, etc.) to help organize your events.
  6. Add a featured image (optional)
    This image appears on the event listing and detail page.
  7. Publish the event
    Click "Publish" to make the event visible on your site.

Event Types

SocietyPress comes with several event types (categories) to help organize your events:

  • Classes — Educational sessions on specific topics
  • Workshops — Hands-on learning experiences
  • General Membership Meetings — Regular society meetings
  • Committee Meetings — Smaller group meetings
  • Leadership Meetings — Board and officer meetings

You can add your own event types by going to Events → Categories in the WordPress admin.

Registration and Time Slots

Some events need members to sign up in advance. SocietyPress supports registration with optional time slots and capacity limits.

Enabling Registration

  1. Edit the event
  2. Find the "Registration" section
  3. Check "Registration required for this event"

Adding Time Slots

Time slots are useful when an event has multiple sessions or limited capacity. For example, a one-on-one research help session might have 30-minute slots throughout the day.

  1. In the "Time Slots" section, click "Add Slot"
  2. Enter the slot details:
    • Start time
    • End time
    • Capacity — How many people can register
    • Description (optional)
  3. Add more slots as needed
No Slots? No Problem!

If you enable registration but don't add time slots, members can still register for the event itself. Slots are only needed when you have multiple sessions or limited capacity.

How Registration Works for Members

When a member views an event that requires registration:

  1. They see the available time slots (if any) with remaining capacity
  2. They click to register for a slot
  3. If the slot is full, they can join the waitlist
  4. The registration appears in their Member Portal under "My Events"

Waitlist

When a time slot reaches capacity, additional registrations go on a waitlist. If someone cancels, the first person on the waitlist is automatically promoted to a confirmed spot (and notified by email).

Recurring Events

For events that happen regularly (weekly meetings, monthly workshops, etc.), you can set up a recurrence pattern instead of creating each event manually.

Setting Up Recurrence

  1. Create or edit the event
  2. Find the "Recurrence" section
  3. Choose a pattern
    • Weekly — Repeats every week on the same day (e.g., every Tuesday)
    • Monthly — Repeats on a specific pattern (e.g., "2nd Tuesday of the month" or "Last Friday")
  4. Set an end date (optional)
    If the recurrence should stop at a certain point, enter an end date. Otherwise, it continues indefinitely.
Monthly Pattern Options

For monthly events, SocietyPress automatically detects the pattern from your event date. If your event is on January 14, 2025 (the 2nd Tuesday), it will recur on the 2nd Tuesday of each month.

Duplicating Events

Need to create a similar event? Instead of starting from scratch, duplicate an existing one and modify it.

  1. Go to the Events list (SocietyPress → Calendar)
  2. Hover over the event you want to copy
  3. Click "Duplicate"
  4. A new draft event is created with all the same details
  5. Edit the date, title, and any other details
  6. Publish the new event

Managing Registrations

As an administrator, you can view and manage event registrations.

Viewing Registrations

  1. Edit the event
  2. Look for the "Registrations" section
  3. You'll see a list of who's registered, including:
    • Member name
    • Time slot (if applicable)
    • Registration date
    • Status (Confirmed or Waitlist)

Canceling a Registration

If a member needs to cancel but can't do it themselves, you can cancel their registration from the admin:

  1. Find the registration in the list
  2. Click "Cancel" next to their name
  3. If there's a waitlist, the next person is automatically promoted

Event Display on Your Site

The SocietyPress theme displays events in several places:

  • Homepage — Shows the next 3 upcoming events
  • Events archive — Lists all upcoming events at /events/
  • Single event page — Full details when clicking on an event

Each event shows:

  • Title and description
  • Date and time
  • Location and address
  • Instructors (if set)
  • Featured image
  • Registration form (if registration is enabled)

Event Emails

SocietyPress can send automatic emails for events:

  • Registration confirmation — When a member registers
  • Waitlist notification — When added to waitlist
  • Promotion from waitlist — When moved from waitlist to confirmed
  • Cancellation confirmation — When a registration is cancelled
Development Mode

When WordPress debug mode is on or you're on localhost, emails are logged but not actually sent. This prevents test emails from going to real members. Check the Email Log in admin to see what would have been sent.
