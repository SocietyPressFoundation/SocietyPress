Installation Guide

This guide walks you through installing WordPress and SocietyPress on your web hosting. Choose the method that works best for you.

Recommended: Use the Installer

The SocietyPress Installer handles everything automatically — downloading WordPress, setting up the database, and installing the plugin and theme. It's the easiest way to get started.

System Requirements

Before you begin, make sure your web hosting meets these requirements:

Requirement Minimum Recommended
PHP Version 8.0 8.2 or higher
MySQL Version 5.7 8.0 or MariaDB 10.4+
WordPress Version 6.0 Latest version
Disk Space 50 MB 100 MB+ (for uploads)

Not sure about your hosting? Most modern web hosting services (GoDaddy, Bluehost, HostGator, SiteGround, etc.) meet these requirements. If in doubt, contact your hosting provider.

Option A: Using the Installer (Recommended)

The SocietyPress Installer is a single file that guides you through the entire setup process. No technical knowledge required!

  1. Download the installer
    Download sp-installer.php from the SocietyPress website or GitHub releases page.
  2. Upload to your hosting
    Use your hosting's File Manager or an FTP program to upload sp-installer.php to the folder where you want your website (usually public_html or www).
  3. Visit the installer in your browser
    Open your web browser and go to:
    https://yourwebsite.com/sp-installer.php
    (Replace "yourwebsite.com" with your actual domain name)
  4. Follow the on-screen steps
    The installer will guide you through:
    • Checking your server requirements
    • Setting up the database
    • Downloading and installing WordPress
    • Installing the SocietyPress plugin and theme
    • Basic configuration
  5. Delete the installer
    Very important! When the installer finishes, delete the sp-installer.php file from your hosting. Leaving it there is a security risk.
Important: Delete the Installer!

After installation is complete, you must delete the installer file. Anyone who finds it could potentially take over your website.

Option B: Manual Installation

If you prefer to install manually, or if the automated installer doesn't work with your hosting, follow these steps.

Step 1: Install WordPress

If you don't already have WordPress installed:

  1. Download WordPress from wordpress.org
  2. Create a database in your hosting control panel (cPanel, Plesk, etc.)
  3. Upload the WordPress files to your hosting
  4. Visit your website and follow the WordPress installation wizard

Many hosting providers offer one-click WordPress installation through tools like Softaculous or Installatron. Check your hosting control panel for these options.

Step 2: Install the SocietyPress Plugin

  1. Download the SocietyPress plugin ZIP file
  2. Log into your WordPress admin dashboard
  3. Go to Plugins → Add New
  4. Click the Upload Plugin button at the top
  5. Choose the ZIP file you downloaded and click Install Now
  6. After installation, click Activate Plugin

Step 3: Install the SocietyPress Theme

  1. Download the SocietyPress theme ZIP file
  2. In WordPress admin, go to Appearance → Themes
  3. Click the Add New button at the top
  4. Click the Upload Theme button
  5. Choose the ZIP file and click Install Now
  6. After installation, click Activate

Verifying Your Installation

After installation, verify everything is working:

  1. Check the plugin is active
    Go to Plugins → Installed Plugins. You should see "SocietyPress" listed and activated.
  2. Check the theme is active
    Go to Appearance → Themes. The SocietyPress theme should show as active.
  3. Check the SocietyPress menu
    You should see a "SocietyPress" menu item in the WordPress admin sidebar with options like Dashboard, Members, Events, etc.
  4. View your website
    Visit your website's front page. You should see the SocietyPress theme design.
What You'll See

After activation, SocietyPress automatically creates its database tables and default settings. You're ready to start configuring your site!

Post-Installation Checklist

After installing SocietyPress, complete these setup tasks:

  • ☐ Configure organization settings (name, address, contact info)
  • ☐ Review and customize membership tiers
  • ☐ Set up navigation menus
  • ☐ Add your logo in the Customizer
  • ☐ Create initial pages (About, Contact, etc.)
  • ☐ Import existing members (if you have them)

See the Getting Started guide for detailed instructions on each of these tasks.
