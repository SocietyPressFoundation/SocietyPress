Managing Members

The Members section is where you manage your society's membership roster. This guide covers everything from adding individual members to bulk importing from a spreadsheet.

The Members List

Go to SocietyPress → Members to see your full member list. From here you can:

  • Search — Find members by name, email, or member number
  • Filter — Show only certain statuses (Active, Expired, etc.) or tiers
  • Sort — Click column headers to sort by name, join date, expiration, etc.
  • Bulk actions — Select multiple members to delete or change status

Member Statuses

Each member has a status that indicates their current standing:

Status Meaning
Active Current, paid member in good standing
Pending Applied but not yet approved or paid
Expired Membership has lapsed and needs renewal
Cancelled Member chose to cancel their membership
Deceased Member has passed away (keeps historical record)

Adding Members Manually

To add a single member:

  1. Go to SocietyPress → Add New Member
  2. Enter basic information
    Required fields are marked with an asterisk (*):
    • First name *
    • Last name *
    • Email address *
    • Membership tier *
    • Status *
  3. Add contact information
    Phone numbers, mailing address, secondary email, etc.
  4. Add genealogy information
    This is what makes SocietyPress special:
    • Surnames — Family names they're researching
    • Research areas — Geographic locations (county, state, country)
  5. Set dates
    • Join date (when they became a member)
    • Expiration date (when membership expires)
  6. Click "Add Member"

Importing Members from a Spreadsheet

If you have an existing member list in a spreadsheet, you can import it into SocietyPress. The importer handles CSV, TSV, and Excel files.

Preparing Your Spreadsheet

Before importing, make sure your spreadsheet:

  • Has a header row with column names
  • Uses consistent formatting (especially for dates)
  • Doesn't have merged cells
  • Is saved as CSV, TSV, or XLSX format
Column Names

The importer tries to automatically match your column names to SocietyPress fields. Using standard names like "First Name," "Last Name," "Email," "Phone," etc. helps the matching work better.

Import Steps

  1. Go to SocietyPress → Import Members
  2. Upload your file
    Click "Choose File" and select your spreadsheet. Click "Upload and Preview" to continue.
  3. Map columns to fields
    The importer shows your columns and tries to match them to SocietyPress fields. Review each mapping and adjust if needed using the dropdowns.
    • Green = matched automatically
    • Yellow = needs your attention
    • Skip = this column won't be imported
  4. Set import options
    Choose what to do with duplicates (skip, update, or create new). Select a default membership tier for records without one.
  5. Start the import
    Click "Import Members" to begin. You'll see progress as records are processed.
  6. Review results
    After import, you'll see a summary of how many records were imported, updated, or skipped (and why).
Test First!

Before importing your entire member list, test with a small sample (5-10 records) to make sure the mapping is correct and data looks right.

Editing Member Information

To edit an existing member:

  1. Go to SocietyPress → Members
  2. Find the member (use search if needed)
  3. Click on their name or the "Edit" link
  4. Make your changes
  5. Click "Update Member"

What You Can Edit

  • All contact information
  • Membership tier and status
  • Join and expiration dates
  • Surnames and research areas
  • Notes and custom fields

Member Photos

Members can have profile photos that appear in the directory and admin screens.

  1. Edit the member's record
  2. Find the "Member Photo" section
  3. Click "Set Photo" to upload an image
  4. Save the member

Photos are automatically cropped to a square and optimized for web display. Recommended size: at least 200×200 pixels.

Genealogy Fields

SocietyPress includes specialized fields for genealogical research that you won't find in generic membership plugins.

Surnames Researched

Track what family names each member is researching. This helps members connect with others who might be researching the same families.

For each surname, you can record:

  • The surname itself
  • Spelling variants (e.g., Smith, Smyth, Smithe)
  • Notes about the line

Research Areas

Geographic areas where a member is conducting research. This helps members find others researching in the same locations.

For each research area, you can record:

  • County
  • State/Province
  • Country
  • Region or other notes

Exporting Members

You can export your member list to a CSV file for use in spreadsheets, mailing lists, or other systems.

  1. Go to SocietyPress → Members
  2. Apply filters (optional)
    If you only want to export certain members (e.g., only Active members), use the filter dropdowns first.
  3. Click "Export"
    Find the Export button above the member list.
  4. Choose what to export
    Select which fields to include in the export.
  5. Download the file
    Your browser will download a CSV file you can open in Excel, Google Sheets, or any spreadsheet program.
Export Respects Filters

The export only includes members that match your current filter settings. To export everyone, make sure no filters are applied.

Linking WordPress User Accounts

Members can optionally have a WordPress user account, allowing them to:

  • Log into the member portal
  • Update their own profile information
  • Register for events
  • Access members-only content

Creating a User Account for a Member

  1. Edit the member's record
  2. Find the "WordPress Account" section
  3. Click "Create User Account"
  4. The member will receive an email with login instructions

Linking an Existing User Account

If a member already has a WordPress account (perhaps they registered on your site), you can link it to their member record:

  1. Edit the member's record
  2. Find the "WordPress Account" section
  3. Search for and select the existing user
  4. Save the member

Bulk Actions

To perform actions on multiple members at once:

  1. Go to the Members list
  2. Check the boxes next to the members you want to affect
  3. Use the "Bulk Actions" dropdown to select an action
  4. Click "Apply"

Available bulk actions include:

  • Delete members
  • Change status (Active, Expired, etc.)
  • Change membership tier
Select All Across Pages

If you have many members, they're shown across multiple pages. Use the "Select all X members" link that appears to select members on all pages, not just the current page.
