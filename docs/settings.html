Settings Reference

This page documents all the settings available in SocietyPress. Access settings by going to SocietyPress → Settings.

Organization Settings

Basic information about your society. This information appears throughout your site and in member communications.

Setting Description
Organization Name Your society's official name (e.g., "Springfield Genealogical Society")
Street Address Physical location of your society or meeting place
Phone Number Contact phone number for the society
Email Address Primary contact email
Operating Hours When your facility is open (if applicable)
Directions How to find your location
Parking Info Parking instructions for visitors
Using Organization Info

You can display organization info on any page using shortcodes:
[sp_address] [sp_phone] [sp_email]

Display Settings

Control how information is displayed in the admin and on your site.

Setting Description Default
Members Per Page How many members to show per page in the admin list 20
Directory Per Page How many members to show per page in the public directory 24
Member Photos Enable or disable member profile photos Enabled
Dashboard Widgets Show SocietyPress widgets on the WordPress dashboard Enabled
Expiring Soon Days Days before expiration to show members in "Expiring Soon" widget 30
Recent Days Days to include in "Recent Signups" widget 30

Membership Settings

Configure how memberships work.

Setting Description
Expiration Model How membership expiration is calculated:
Calendar Year - All memberships expire December 31
Rolling - Expires X months after join/renewal date
Portal Enabled Allow members to access the self-service portal

Email Settings

Configure how emails are sent from SocietyPress.

Setting Description
From Name Name that appears in the "From" field of outgoing emails
From Email Email address used as the sender
Admin Email Where admin notifications are sent
Email Deliverability

For reliable email delivery, consider using an SMTP plugin like "WP Mail SMTP" to send emails through a proper email service (Gmail, SendGrid, etc.) rather than your web server.

Breadcrumb Settings

Configure the breadcrumb navigation on your site.

Setting Description Default
Separator Character between breadcrumb items /
Home Icon Show a home icon instead of "Home" text Enabled
Home Text Text for the home link (if icon disabled) Home

Leadership & Committees

SocietyPress includes features for tracking your organization's leadership structure. These are managed from separate menu items.

Leadership Positions

Go to SocietyPress → Leadership to manage:

  • Positions — Define roles like President, Vice President, Secretary, etc.
  • Position Holders — Assign members to positions with term dates
  • Board vs. Officer — Flag positions as board members or officers

Committees

Go to SocietyPress → Committees to manage:

  • Committee definitions — Create committees (Programs, Membership, etc.)
  • Committee members — Assign members with roles (Chair, Vice-Chair, Member)
  • Standing committees — Flag permanent vs. temporary committees

Member Levels (Tiers)

Go to SocietyPress → Member Levels to configure membership tiers.

For each tier, you can set:

Field Description
Name Display name (e.g., "Individual", "Family")
Slug Internal identifier (auto-generated from name)
Price Annual dues amount
Duration Membership length in months (0 = lifetime)
Max Members How many people a membership covers (1 for individual, 2 for family, etc.)
Description Optional description shown on join forms

Default Tiers

SocietyPress creates these tiers automatically on installation:

  • Individual — $35/year, 1 member
  • Family — $45/year, 2 members
  • Student — $20/year, 1 member
  • Lifetime — $500 one-time, never expires
  • Institutional — $50/year, for libraries/archives

Modify these to match your organization's actual dues structure.

Shortcodes

SocietyPress provides shortcodes you can use in pages and posts to display dynamic content.

Shortcode Output Attributes
[sp_address] Organization address link="yes" for Google Maps link
[sp_phone] Organization phone link="yes" for clickable tel: link
[sp_email] Organization email link="yes" for mailto: link (default)
[sp_website] Organization website link="yes", text="Visit us"
[sp_year] Current year (none)
[sp_organization] Organization name (none)
[sp_modified] Page last modified date format="F j, Y"
[societypress_contact] Full contact block show_form, show_hours, etc.

Example Usage

Contact us at [sp_email] or call [sp_phone link="yes"].

Visit us at:
[sp_address link="yes"]

© [sp_year] [sp_organization]. All rights reserved.

Menu Placeholders

In your navigation menus, you can use {{organization_name}} in menu item labels. SocietyPress automatically replaces this with your organization's name from settings.

This is useful for menu items like "About {{organization_name}}" that automatically update if you ever change your organization's name.
