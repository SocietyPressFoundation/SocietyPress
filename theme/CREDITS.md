## Fonts

The theme uses system font stacks for optimal performance and native OS appearance:

- **Sans-serif:** -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, "Helvetica Neue", Arial, sans-serif
- **Serif:** Georgia, Cambria, "Times New Roman", Times, serif
- **Monospace:** ui-monospace, SFMono-Regular, "SF Mono", Menlo, Consolas, monospace

No external font files are loaded, ensuring fast page loads and GDPR compliance.
